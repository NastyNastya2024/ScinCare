# ScinCare - Science-backed AI Cosmetologist

A modern landing page for a skincare AI assistant application, inspired by Lóvi.

## Features

- Responsive design
- Modern animations
- Interactive FAQ section
- Smooth scrolling navigation
- Mobile-friendly layout
- Three main superpowers:
  - Face scanning with 11+ skin issues tracking
  - Cosmetics scanner with fit score
  - AI assistant chat

## Setup

1. Clone the repository
2. Open `index.html` in a browser
3. Or use a local server:
   ```bash
   python -m http.server 8000
   ```
   Then open `http://localhost:8000` in your browser

## Structure

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Modern CSS animations

## Customization

You can customize the colors by modifying CSS variables in `styles/main.css`:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    /* ... */
}
```
